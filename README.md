🎓 Student Management System in C++

 📌 Project Overview

This is a console-based Student Management System developed using C++. It helps manage student records efficiently with file handling, allowing persistent storage of data.

 🚀 Features

* Add new student record
* Display all students
* Search student by ID
* Update student details
* Delete student record
* Data stored using file handling
* Input validation and error handling

 🛠 Technologies Used

* C++ Programming Language
* File Handling (fstream)
* Classes and Objects

 ▶️ How to Run

1. Compile the program:
g++ main.cpp -o student
2. Run the program:
./student
